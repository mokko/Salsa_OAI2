Salsa OAI - Minimalistic OAI data provider with Dancer

Quick Tests

Test with http client of your choice, e.g. web browser, Tim Brody's oai_browser.pl or curl

Some shortcuts:

  • Identify (describe repository)
  • ListMetadataFormats (show metadataFormats supported by repository, identifier is an optional argument)
  • ListIdentifiers (metadataPrefix=mpx) (list all identifiers in the repository)
  • ListIdentifiers (metadataPrefix=mpxXX (list all identifiers in the repository)
  • ListRecords () (list all records in specified metadataFormat; should fail)
  • GetRecord() (a incomplete GetRecord, should fail)
  • GetRecord(identifier=spk-berlin.de:EM-objId-100064, metadataPrefix=mpx) (a GetRecord, wtihout transformation, should pass)
  • GetRecord(identifier=spk-berlin.de:EM-objId-100064, metadataPrefix=oai_dc) (a GetRecord with transformation, should pass)
  • ListRecords (metadataPrefix=mpx) (list all records in specified metadataFormat; should eventually pass)
  • ListSets () (list all sets ; should eventually pass)
