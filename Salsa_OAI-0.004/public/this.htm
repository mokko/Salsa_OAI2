Salsa OAI - Perl OAI data provider with Dancer

Data

This implementation of Salsa OAI provides information on some collections of the Ethnological Museum in Berlin. Currently the focus is on musical instruments (see set MIMO). It was set up for the MIMO project, see http://www.mimo-project.eu.

Changes

For a list of changes to the data provider, see Salsa OAI and HTTP::OAI::DataProvider on github. This list mentions changes to the data displayed.
