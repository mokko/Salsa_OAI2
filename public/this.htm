Salsa OAI - Minimalistic OAI data provider with Dancer

About this implementation of Salsa_OAI

This OAI data provider offers data on musical instruments in the Ethnological Museum in Berlin. It was set up for the MIMO project, see http://www.mimo-project.eu.
