Salsa OAI - Perl OAI data provider with Dancer

About Salsa OAI

Salsa OAI is a simple OAI 2.0 data provider. For more see https://github.com/mokko/Salsa_OAI2

Currently supported

  • All six OAI v2.0 verbs should work.
  • All errors as described in the specification work.
  • ResumptionToken works

Not (yet) supported

  • Deleted records. I currently have no reliable info on deleted records in my source data, so I have a hard time implementing it, but it should almost work.

Components

Salsa OAI is a small frontend to HTTP::OAI::DataProvider (on Github.com). Currently there is only one sqlite database backend. In the past I experimented with a file-based solution, but this easily leads to memory shortages.
