Salsa OAI - Perl OAI data provider with Dancer

About Salsa OAI

Salsa OAI is a simple OAI 2.0 data provider. It uses Dancer and a specific metadata format called mpx. Salsa OAI is a specific implementation of HTTP::OAI::DataProvider. You could regard Salsa OAI as an example of how to set up your own HTTP::OAI::DataProvider.

HTTP::OAI::DataProvider tries not to depend on perl Dancer, it is metadata format agnostic and modularized. Currently it works with an easy-to-install sqlite backend, but it should be fairly easy to write another backend.

Salsa OAI and HTTP::OAI::DataProvider are written in 2010/2011 by Maurice Mengel for the https://github.com/mokko/Salsa_OAI2
